In position : relative the current block or element moves according to the X&Y Axis.

In position : absolute, the current block or element moves according, only if parent is non-static(relative etc) if recent parent is static it checks for grand parent and this goes on until it finds any of parent as non-static.
If none of the parent is non static then it moves according to the body.
