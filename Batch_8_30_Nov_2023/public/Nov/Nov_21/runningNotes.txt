    id - To add unique identity to html element

    name - To group set of reletave elements with same name attribute value 

    class - To inject single/multiple css classes to any html element 

    title - To add a  title content to any html element, where the content will automatically shown 
            as a tool tip, while user mouseovers on the element. 
    
    style - Used to inject single/multiple css properties to html element 

    src - Used to provide the source/path of resource 

    href - Used to provide the hyperlink path of page 

    alt - Using which we can add the alternative text content which willbe shown automatically if the provided image path is not available
